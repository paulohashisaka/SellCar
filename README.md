# SellCar Landing Page

Landing page responsiva criada com React, Vite e Tailwind CSS.

## Requisitos

- Node.js 20.19+ ou 22.12+
- npm

## Instalação

```bash
npm install
```

## Desenvolvimento

```bash
npm run dev
```

Abra o endereço exibido pelo Vite no terminal.

## Build de produção

```bash
npm run build
npm run preview
```

## Personalização

- Substitua o e-mail demonstrativo em `src/components/ContentSections.jsx`.
- Edite os depoimentos no array `testimonials` do mesmo arquivo.
- Atualize textos, preços e condições conforme os dados oficiais da SellCar.
